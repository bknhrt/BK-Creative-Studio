# Cartao digital estilo Linktree

Arquivos:

- `index.html`: estrutura do cartao e links.
- `styles.css`: visual, cores e responsividade.
- O fundo animado fica em `.animated-bg`.
- O modo escuro automatico fica em `@media (prefers-color-scheme: dark)`.

Para personalizar:

1. Troque `@seuperfil` pelo usuario do Instagram.
2. Troque `Seu Nome ou Marca` e a bio curta no topo.
3. Atualize os links dos botoes:
   - Instagram: `https://instagram.com/`
   - WhatsApp: `https://wa.me/5500000000000`
   - E-mail: `mailto:contato@email.com`
   - Portfolio ou catalogo: `#`
4. Substitua `SP` pelas iniciais ou por uma foto, se quiser.
5. Troque `seulink.com.br/seuperfil` pelo link final da bio ou do QR Code.

Para ver no navegador, abra o arquivo `index.html`.
